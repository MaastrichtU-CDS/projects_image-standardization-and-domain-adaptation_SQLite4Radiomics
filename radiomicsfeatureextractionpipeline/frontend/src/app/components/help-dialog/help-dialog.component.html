Modify Parameter File

Opens yaml parameter file and let's you edit and save it, as well as restore to default or previous version.

Modify Configuration File

Opens the config ini file and let's you edit and save it, as well as restore to default or previous version.

Recalculate Features

Request radiomics pipeline extraction on all currently stored DICOM data.

Download CSV Files

Downloads CSV files currently in storage as zip archive.

Download Log Files

Downloads log files currently in storage as zip archive.

Set ROI Priority

Run the ROI priority setter, where you can set the priority for each region of interest to be used in radiomic calculations. Give an interger value, 1 being the highest priority and 10 being lowest priority. If no priority has of yet been given to a specific ROI, you will see the value -1.

Reinitialize Radiomics Tables

Deletes all radiomics tables and creates them again.

WARNING: All data stored in those tables will get deleted!

Delete Calculation

Allows you to pick a calculation that was previously executed, by picking the day and time of said calculation, and delete it.
